# 🚗 UTPGO – Campus Ride Booking App

A campus ride-booking Android application developed using **Kotlin**, **Java** and **Android Studio**. This project was originally designed in **Figma** and later implemented as a functional Android application.

## ✨ Features
- User login
- Ride booking
- Pickup and destination selection
- Booking details
- Clean and user-friendly interface

## 🛠️ Technologies
- Kotlin
- Java
- Android Studio
- XML
- Figma

## 📖 Overview
UTPGO is designed to simplify campus transportation by providing students with an intuitive platform to book rides within and outside the university campus. This project demonstrates Android development, UI implementation, and mobile application design
